# Stock Market Dashboard

This is a simple React application that allows users to get information about stocks of well-known company. We have used a free version of API which have limit of 25 requests per day. Please use vpn if you encounter such error.

## Setup

1. **Clone the repository:**

    ```bash
    git clone https://github.com/monurajputrko/Stock-Market.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd Stock-Market
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```

## Usage

1. **Run the development server:**

    ```bash
    npm start
    ```

2. **Open the application in your browser:**

    The development server should start running at `http://localhost:3000`. Open this URL in your web browser to view the application.
